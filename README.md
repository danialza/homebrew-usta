# homebrew-usta

Homebrew tap for [Usta](https://github.com/danialza/Usta) — your AI engineering team, native on macOS.

## Install

```sh
brew install --cask danialza/usta/usta
```

Then open **Usta** from Applications. That's it — the cask clears the Gatekeeper quarantine flag for you.

## Update

```sh
brew upgrade --cask usta
```
